# Face Mask Detection Project

## Overview
This project is an AI-based Face Mask Detection System built using Deep Learning and CNN.

## Technologies Used
- Python
- TensorFlow
- Keras
- Google Colab
- CNN

## Features
- Detects whether a person is wearing a mask or not
- Image-based prediction
- Trained using custom dataset

## Project Workflow
1. Dataset creation
2. Image preprocessing
3. CNN model training
4. Prediction testing
5. Model saving

## Output
The model predicts:
- Mask Detected
- No Mask Detected

## Future Improvements
- Live webcam detection
